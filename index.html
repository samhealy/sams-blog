Portfolio Project: Utilizing jQuery

My goal for this project was to design and develop a unique personal website. I started by adding a home page that displays what the site has to offer: an about page, a portfolio page, and a blog page.

One of the major features of this website is the interactive sidebar. Upon clicking on the home page, the screen slides over and collapses into a sidebar, revealing a page underneath. When you hover over the sidebar, the menu options pop in and out. This feature was implemented using jQuery.

I chose to make this site a single page application as opposed to linking to separate pages. All of my HTML is in one document, and I utilized jQuery functions hide different parts of the document at specific times, giving the illusion of having multiple pages.

The last thing I did was to add filler content to my site. Thank you for your interest!
